# HerbCalc: Tools for Herbalists

Open-source herbalism toolkit with calculators, converters, and reference tools for herbal formulation. Built with Gradio/Python.

Developed in the context of herbalism education at [SOIH (School of Integrative Health)](https://soih.edu/).

## Features

- **Unit Conversion** -- Metric, imperial, apothecary, and household unit translation with historical and pharmacy shorthand support.
- **Solvent Calculations** -- Menstruum calculator, Pearson's square alcohol dilution, solvent affinity reference.
- **Formulation** -- Multi-herb formula builder, dosage scaler, dry-extract converter.
- **Preparation Calculators** -- Infusion/decoction, syrup, and salve/balm calculators.
- **Safety & Quality** -- Herb-drug interaction checker and batch log generator.
- **Reference** -- Herb monograph viewer, pharmacy abbreviation decoder, glossary.

## Setup

```bash
pip install -r requirements.txt
python app.py
```

The app runs on `localhost:7860` by default.
